## Chinese fonts

中文模板与英文模板的区别仅有两行——使用中文时仅需反注释以下两行，模板中已默认启用，第一次编译时耗时相对较长(引入了外部中文字型)，耐心等待下。

```latex
\usepackage{zh_CN-fonts} % Simplified Chinese Support using system fonts
```
